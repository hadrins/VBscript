VB scripts

Most of these run from a command prompt. as it is hard to tell what the default script engine is on random machines. 

Dell_service_tag.vbs   This read the Dell Server Tag and print the message out. 

Dirscan.vbs  give it a path and it will list the size ot the subdirectories.

Updates.vbs   this will tell you the Windows updates settings. 

User_account_info.vbs   This will tell you the user account information.
